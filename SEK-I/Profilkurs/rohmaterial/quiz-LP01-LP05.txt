==========================================================================

Wofür steht EVA?
Eingabe, Verarbeitung, Ausgabe
Entwicklung, Verifikation, Anwendung
Eingabe, Verschlüsselung, Ausgabe

Was ist ein Algorithmus?
eine eindeutige, endliche Schrittfolge zur Lösung eines Problems
ein beliebiger Name für ein Programm

ein Gerät zur Dateneingabe

Was macht eine Verzweigung in einem Algorithmus?

Was macht eine Schleife in einem Algorithmus?

sie trifft eine einmalige Entscheidung

LP04 — Schleifen & Verzweigungen: ein Mini-Spiel
==========================================================================

Wozu dient eine Endlosschleife beim Abfragen der Tastatur?
damit ständig geprüft wird, ob eine Taste gedrückt wurde
damit das Programm nach einem Tastendruck sofort endet
damit die Tastatur ausgeschaltet wird
sie hat damit nichts zu tun

Was passiert bei der Verzweigung "falls Figur berührt Rand"?
die Figur reagiert (z. B. dreht ab), wenn die Bedingung erfüllt ist
die Figur verschwindet immer sofort
das Spiel startet automatisch neu
nichts, Verzweigungen wirken nicht auf Figuren

Wozu dient eine Variable wie "Punkte" in einem Spiel?
sie speichert einen Zustand (den Punktestand), der sich über die Zeit verändert
sie zeichnet nur die Spielfigur
sie ist nur für die Farbe der Bühne zuständig
Variablen werden in Scratch nicht benötigt

Womit erkennst du eine Kollision zwischen zwei Figuren?
mit einem "wird berührt?"-Block als Bedingung
nur durch genaues Hinschauen ohne Code
mit einer Endlosschleife allein
das ist in Scratch nicht möglich

Was unterscheidet eine Schleife von einer Verzweigung grundsätzlich?
eine Schleife wiederholt, eine Verzweigung entscheidet einmalig zwischen Möglichkeiten
beide machen exakt dasselbe
eine Verzweigung wiederholt, eine Schleife entscheidet

damit die Punktzahl nie steigen kann
