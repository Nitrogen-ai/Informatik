Schwer
Schreibe einen Sketch, der die interne LED (Pin 13) in einer Dauerschleife im Sekundentakt blinken lässt.
Lösung
void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}

Schwer
Großes Beispiel (RSA): Finde die beiden Primfaktoren von n = 4757.
Lösung
